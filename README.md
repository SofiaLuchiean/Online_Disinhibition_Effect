## Formal Model of the Online-Disinhibition-Effect

This repository contains the material and source code for a formal model of the Online Disinhibition Effect (Suler, 2004). 

The base of this formal model was developed as part of the [Formal Modeling of Psychological Theories course](https://nicebread.github.io/FOMO-Psy/) at LMU Munich by Amelie Schlieben, Lara Haas, Martje Buss and me, and can be found [here.](https://github.com/amelieschlieben/Online-Disinhibition-Effect)
Additionally, this repository displays my individual contribution to expanding that baseline formal model (which is to be graded).

Supplementary tables (Construct Source Table, Variable Table and Relationships Table) can be found [in this Google doc.](https://docs.google.com/document/d/1bz6I23mhWh55Sqn0t3X7c7IqWmTZcL9JOExeKnv2ro8/edit?usp=sharing)
